BankSight is a Banking Management System Dashboard built using Python, Streamlit, and MySQL.
It allows users to manage banking data such as customers, accounts, transactions, loans, branches, and support tickets through a simple and interactive dashboard.
The project demonstrates CRUD operations, database integration, and data visualization in a real-world banking scenario.
Features:
  1.Interactive banking dashboard
  2.CRUD operations
  3.MySQL database integration
  4.Basic analytics and filtering
  5.Customer and account management

Technologies Used:
  1.Python
  2.Streamlit
  3.MySQL
  4.Pandas
  5.PyMsql
